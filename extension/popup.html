How to Use Flow Control

Simple Mode

Set a single speed for all videos on the current site.

  • Use slider or input to set speed
  • Speed remembered per website
  • Works on all video players

Ranges Mode

Set different speeds based on video duration.

  • Perfect for different content types
  • Short videos → faster speeds
  • Long videos → comfortable speeds

Shortcuts

[ ]
Decrease/increase speed
'
Hold for temporary speed

Tips

  • Settings save automatically
  • Each website has separate settings
  • Use "Off" to disable completely
